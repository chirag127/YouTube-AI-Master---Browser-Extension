# Changelog

## [1.0.0] - 2025

### Added

-   AI-powered video summarization with Gemini
-   Segment classification (sponsors, intros, highlights)
-   Auto-skip functionality for sponsors/intros
-   Interactive chat with video content
-   Multilingual support (40+ languages)
-   FAQ generation
-   Comment sentiment analysis
-   Transcript extraction with multi-method fallback
-   History management
-   Keyboard shortcuts
-   Service worker keep-alive mechanism

### Features

-   3 summary lengths (Short/Medium/Detailed)
-   Clickable timestamps
-   Visual segment markers
-   Export transcripts
-   Customizable settings
-   Privacy-focused (no tracking)

### Technical

-   Manifest V3 architecture
-   Modular ES6 codebase
-   Invidious API integration
-   Latest Gemini models support
-   Efficient caching
-   Error handling improvements
